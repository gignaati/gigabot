## Reporting a Vulnerability

The Giga Bot team takes security seriously. If you discover a security vulnerability, please report it responsibly.

**Do not create a public GitHub issue for security vulnerabilities.**

Please send a detailed report to: **[support@gignaati.com](mailto:support@gignaati.com)**

Include the following in your report:
- A clear description of the vulnerability
- Steps to reproduce the issue
- The potential impact and severity
- Any suggested mitigations or fixes

We will acknowledge your report within **48 hours** and aim to release a fix within **7 days** for critical issues. We will credit responsible disclosures in the release notes.
